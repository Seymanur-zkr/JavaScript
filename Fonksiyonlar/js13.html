Fonksiyon Nedir

Ana program tarafından gerektiğinde çalıştırılan program parçacıklarına
fonksiyon denir.Program içinde bazı işlemleri sık sık yapıyorsanız
her işlem için kod yazmak yerine bu kodu bir defa yazar ve fonksiyon olarak kaydederiz.
Daha sonrada gerektiğinde bu kodları/fonksiyonları çağırıp çalıştırırız.
Fonksiyon oluştururken aşağıdaki kalıba uygun yazılır ;

fuction fonksiyon_adı(varsa parametreler)
{
kodlar;
return değişken;
}



Fonksiyonlarımızı iki şekilde çalıştırabiliriz;
Kodların arasına yazarak:Çalıştırmak istediğiniz fonksiyonun adını yazarak
herhangi bir fonksiyonun çalışmasını sağlayabilirsiniz.
Bir olayla tetikleyerek: Belirli bir olay gerçekleştiğinde belli bir Javascript
fonksiyonunun çalışmasını sağlayabilirsiniz.Örneğin fareyle bir butona
tıklandığında belirli bir fonksiyonun çalışmasını sağlayabilirsiniz.

Değer Döndürmeyen Fonksiyonlar

Bu tür fonksiyonlar ana program tarafından çağırılarak çalıştırılır fakat
ana programa bir değer göndermezler/döndürmezler. O nedenle bu
tür fonksiyonlara " değer döndürmeyen fonksiyon " denir.

function merhaba()
{
document.write("merhaba");
}
merhaba();

Tek Parametreli Değer Döndürmeyen Fonksiyonlar

Parametre fonksiyona ana program tarafından gönderilen değerdir.
Fonksiyon ana programdan gelen değeri alarak bir değişken gibi kullanılabilir.

function merhaba(ad)
{
document.write("merhaba"+ad+"nasılsın?");
}
merhaba("Ethan Winters");

Çok Parametreli Değer Döndürmeyen Fonksiyonlar

Aşağıdaki fonksiyona ana programdan iki değer gönderilir.
gönderilen değerleri toplar ve ekrana yazar.Ana programa her hangi bir değer döndürmez.


function topla(s1,s2)
{
document.write("Toplam:"+topla);
}
topla(3,5);

Değer Döndüren Fonksiyonlar

Bu tür fonksiyonlar belli bir kodu çalıştırırlar ve kodun çalışması
sonucunda ana programa değer döndürürler. Parametresiz, tek veya çok parametreli olabilirler.
function kare(x)

{
return x*x;
}
document.write(kare(2));
document.write(kare(5));

Tek Parametreli Değer Döndüren Fonksiyonlar

Parametre fonksiyona ana program tarafından gönderilen değerdir.
Fonksiyon ana programdan gelen değeri alarak işlem yapar.
Aşağıdaki örnekte kupal() fonksiyonu tanımlanmıştır.
Fonksiyon ana program tarafından belirtilen değerin küpünü hesaplar
ve ana programa döndürür.

function kupal(x)
{
return x*x*x;
}
var s=Number(prompt("sayı:"));
document.write(kupal(s));

Çok Parametreli Değer Döndüren Fonksiyonlar

Ana programdan fonksiyona bir veya daha fazla değer yani parametre
gönderilebilir. Birden fazla değer gönderilecekse parametreler
parantez içinde araya virgül konularak yazılır.

function topla(a,b)
{
var toplam=a+b;
return toplam;
//return a+b de yazabilirdik.
}
document.write(topla(2,5));



Değişken Ömrü ve Kapsamı

Bir blok{} içinde tanımlanan bir değişken sadece o blok içinde kullanılabilir.
Bir bloğun içindeki komutlar çalıştıktan sonra o blok içinde tanımlanan değişkenler
bellekten silinir. Eğer bu değişkenleri başka bir blok içinde de kullanırsanız
hatalı sonuç verecektir.



Küresel Değğişkenler

Eğer bir değişkenin diğer bloklar tarafından da kulalnılmasını istiyorsanız bu
değişkeni küresel olarak tanımlarsınız.Küresel/Global olarak tanımlamak istediğiniz
değişkeni script tagı içine yazmalısınız.Yerel değişkenler içinde bulundukları
bloğun kodları çalıştıktan sonra bellekten silinirken küresel değişkenler bellekten
ancak program sonlandığında silinir.
