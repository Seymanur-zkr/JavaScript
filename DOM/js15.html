DOM(Document Object Model) Nedir?

DOM bize tarayıcı ve HTML komutları arasındaki ilişki ve hiyeraşi konusunda bilgi veren bir modeldir.
bu model sayesinde HTML sayfasındaki nesneler/komutlar seçilip işlem yapılabilir.
DOM'da her HTML komutu bir nesne olarak davranır. Bu özellikten faydalanarak HTML komutlarında aşağıdaki işlemler yapılabilir.
Sayfaya HTML kodlarının eklenmesi veya silinmesi
HTML komutlarının içeriklerinin değiştirilmesi
HTML komutlarının özelliklerinin değiştirilmesi
CSS kodlarının değiştirilmesi
HTML olaylarının tetiklenmesi


Document Nesnesi

Document nesnesi tarayıcıdaki HTML sayfasını temsil eder.Document nesnesini kullanarak
sayfadaki resim veya form elemanı gibi tüm nesnelere erişebiliriz.
HTML komutlarının seçimi için aşağıdaki yöntemler kullanılabilir:
document.getElementById(id) //ID özelliği ile seçilir
document.getElementsByTagName(name) //Tag adına göre seçilir
document.getElementsByClassName(name) //Class adına göre seçilir

document.getElementsByTagName("p"); //Tüm p taglarını/komutlarını bul/seç demektir.
var x = document.getElementById("main"); //Önce ID değeri “main” olan elemanı seç, daha demektir.
var y = x.getElementsByTagName("p"); //sonra da onun içindeki p taglarını seç demektir.
var x = document.getElementsByClassName("intro"); //Sınıf/class adı “intro” olan komutu seç ve x değişkenine ata demektir.
var x = document.querySelectorAll("p.intro"); //CSS seçici ile komut seçmek için kullanılır. Class özelliği “intro” olan paragrafları seç demektir.

HTML Komutlarının İçeriğini Değiştirmek

HTML komutlarının içeriğini veya özelliğini değiştirmek için aşağıdaki yöntemler kullanılabilir:
komut.innerHTML=içerik
komut.özellik=değer
komut.setAttribute(özellik,değer)
komut.style.property=yeni_stil

HTML Olayları (Eventhandler)

HTML DOM kullanılarak belirli olaylar meydana geldiğinde belirli komutların çalışmasını sağlayabilirsiniz.
Örneğin fare ile bir butona tıklandığında (click olayı) belli bir fonksiyonun çalışmasını sağlayabilirsiniz.
Ya da sayfa açıldığında (onload olayı) belirli kodların otomatik olarak çalışmasını sağlayabilirsiniz.
Olayları kullanarak yapılan programlama mantığına Olay Güdümlü Programlama denir. Bu mantık sadece
JavaScript dilinde değil, C# gibi diğer dillerde de kullanılır.Bazı açıklamalar;
onload // Sayfanın yüklenmesi/açılması bittiğinde “onload” olayı meydana gelir.
onunload // Sayfanın kapanması
onresize // Sayfa boyutu değiştiğinde
onerror // Sayfada bir JavaScript hatası olduğunda
onfocus // Bir form elemanı aktif olduğunda
onblur // Bir form elemanı pasif olduğunda
onchange // Form elemanının değeri değiştiğinde
onsubmit // Form onaylandığında
onreset // Form reset edildiğinde
oninvalid // Forma girilen eleman geçerli değilse
oninput // Forma veri girilirse
onclick // Bir elemana tıklandığında
ondblclick // Bir elemana çift tıklandığında
ondrag // Bir eleman sürüklendiğinde
onmouseover // Bir elemanın üzerine fare geldiğinde
onmouseout // Fare eleman dışına çıktığında
